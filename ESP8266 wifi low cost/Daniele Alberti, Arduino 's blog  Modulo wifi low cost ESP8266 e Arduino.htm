Modulo wifi low cost ESP8266 e Arduino

Ciao a tutti!
oggi vi parlerò del modulo wifi ESP8266.

Questo modulo è davvero fantastico infatti ci permette di dotare i nostri progetti di una connessione wifi con un costo ridicolo!
Grazie a questo oggettino possiamo catapultarci nell' Internet of Things,
infatti con un prezzo che va dai 2 ai 5 euro potremo finalmente far twittare la nostra pianta quando ha sete (sto scherzando, ci sono applicazioni molto più utili ma adesso non mi vengono in mente ;) )

Il modulo in questione è questo:





Come vedete è anche molto piccolo, quindi è facilmente collocabile all'interno dei nostri progetti

Quello che faremo adesso è un semplice esempio per capire il funzionamento di questo modulo,
in pratica sfrutteremo il modulo ESP8266 per stabilire una connessione wifi diretta tra pc e modulo dopo esserci connessi, e aprendo la pagina html che creeremo, potremo inviare comandi all'Arduino premendo i pulsanti direttamente da un browser sul nostro pc.

Ecco lo schema di collegamento, il secondo Arduino è opzionale, serve solo per leggere le informazioni provenienti dal modulo ESP8266:



Ed ecco lo sketch da caricare,
è un ottimo esempio che ho trovato su internet,
l'ho tradotto, testato ed ho fatto delle piccole modifiche per adattarlo al mio modulo in quanto alcuni ESP8266 hanno velocità di comunicazione (baud rate) diverse.


  1. #include <SoftwareSerial.h>
  2. #define DEBUG true
  4. SoftwareSerial mySerial(9,10); //creazione di una porta seriale software per il collegamento con l'atra Arduino (facoltativa, serve solo per il debugging)
  5. void setup()
  6. {
  7.   Serial.begin(115200); // inizializza la comunicazione con il modulo ESP8266 (cambiare la velocità se quella del tuo modulo è differente, di solito è 115200 o 57600 o 9600.
  8.   mySerial.begin(9600); // inizializza la comunicazione con l'atra Arduino (facoltativa, serve solo per il debugging)
  10.   pinMode(11,OUTPUT);
  11.   digitalWrite(11,LOW);
  13.   pinMode(12,OUTPUT);
  14.   digitalWrite(12,LOW);
  16.   pinMode(13,OUTPUT);
  17.   digitalWrite(13,LOW);
  19.   sendData("AT+RST\r\n",2000,DEBUG); // resetta modulo
  20.   sendData("AT+CWMODE=2\r\n",1000,DEBUG); // configura come access point
  21.   sendData("AT+CIFSR\r\n",1000,DEBUG); // ottieni un indirizzo ip
  22.   sendData("AT+CIPMUX=1\r\n",1000,DEBUG); // configura per connessioni multiple
  23.   sendData("AT+CIPSERVER=1,80\r\n",1000,DEBUG); // avvia server sulla porta 80
  24. }
  26. void loop()
  27. {
  28.   if(Serial.available()) // check if the esp is sending a message
  29.   {
  30.     if(Serial.find("+IPD,")) //Cerca nella comunicazione seriale la stringa di testo "+IPD," questa è una stringa che viene inviata quando premi il bottone nella pagina html
  31.     {
  32.       delay(1000);
  33.       int connectionId = Serial.read()-48; //Ottieni il connection id, sottrai 48 perchè nella tabella ASCII lo zero è il 48.
  34.       Serial.find("pin="); // adesso cerca "pin="
  35.       int pinNumber = (Serial.read()-48)*10; //leggi il primo numero e moltiplicalo per 10, es per il pin 13 leggi 1 e moltiplicalo per 10
  36.       pinNumber += (Serial.read()-48); //leggi il secondo numero e sommalo al primo, es per il pin 13 leggi 3 e sommalo a quello che avevamo prima (10)
  38.       digitalWrite(pinNumber, !digitalRead(pinNumber)); //cambia di stato il pin 13 (impostando l'opposto di quello che c'è sul pin 13 stesso)
  40.       // componi il comando di chiusura da inviare al modulo
  41.       String closeCommand = "AT+CIPCLOSE=";
  42.       closeCommand+=connectionId; // attacca il connection id
  43.       closeCommand+="\r\n";
  44.       sendData(closeCommand,1000,DEBUG); // invia il comando di chiusura
  45.     }
  46.   }
  47. }
  49. /*
  50. Funzione usata per inviare i dati al modulo ESP8266
  52. Parametri: command - il dato/comando da inviare
  53.            timeout - il tempo massimo di attesa per una risposta
  54.            debug - attiva o disattiva l'invio di dati per il serial monitor (true = si, false = no)
  55. Returns:   la risposta dell' esp8266 (se c'è una risposta)
  56. */
  58. String sendData(String command, const int timeout, boolean debug)
  59. {
  60.   String response = "";
  62.   Serial.print(command); // invia il comando all' esp8266
  64.   long int time = millis();
  66.   while((time+timeout) > millis())
  67.   {
  68.     while(Serial.available())
  69.     {
  70.       char c = Serial.read(); // componi la risposta coi caratteri ricevuti dal modulo
  71.       response+=c;
  72.     }  
  73.   }
  75.   if(debug)
  76.   {
  77.     mySerial.print(response);
  78.   }
  80.   return response;
  81. }


Sketch da caricare nel secondo Arduino (opzionale, serve solo per il leggere le informazioni provenienti dal modulo per vedere che non ci siano errori)

Questo invece è il codice Html per creare la pagina da dove controllare l'Arduino,
quello che dovete fare è:

- creare una nuova cartella
- creare all'interno della cartella un nuovo documento di testo
- copiare le righe di codice che vedete qui sotto nel documento e salvarlo
- rinominare il file e modificarne l'estensione in ".html" (ad esempio "myesp8266control.html")
- andare in questa pagina e cliccare col tasto destro, poi "salva con nome" e salvate il file nella cartella creata prima


  1. <html>
  2.         <head>
  3.                 <title>ESP8266 LED Control</title>
  4.         </head>
  5.         <body>
  6.        
  7.         <!-- in the <button> tags below the ID attribute is the value sent to the arduino -->
  8.        
  9.         <button id="11" class="led">Toggle Pin 11</button> <!-- button for pin 11 -->
  10.         <button id="12" class="led">Toggle Pin 12</button> <!-- button for pin 12 -->
  11.         <button id="13" class="led">Toggle Pin 13</button> <!-- button for pin 13 -->
  12.                
  13.         <script src="jquery.min.js"></script>
  14.         <script type="text/javascript">
  15.                 $(document).ready(function(){
  16.                         $(".led").click(function(){
  17.                                 var p = $(this).attr('id'); // get id value (i.e. pin13, pin12, or pin11)
  18.                                 // send HTTP GET request to the IP address with the parameter "pin" and value "p", then execute the function
  19.                                 $.get("http://192.168.4.1:80/", {pin:p}); // execute get request
  20.                         });
  21.                 });
  22.         </script>
  23.         </body>
  24. </html>

Adesso dovrete cercare il modulo tra le connessioni wifi disponibili, connettervi, e aprire la pagina html che avete creato, premendo il pulsante relativo al pin 13 il led presente sul vostro Arduino dovrà cambiare di stato, quindi si accenderà o spegnerà.
